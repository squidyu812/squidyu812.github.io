## Normal

## Homebrew

OSX 套件管理軟體，下面列出一些比較常見的指令：

```
$ brew search [TEXT|/REGEX/]
```

* 搜尋 Homebrew 套件

```
$ brew list
```

* 用來檢查使用者透過 Homebrew 已安裝的 Package 有哪些 

```
$ brew info [FORMULA...]
```

* 套件資訊

```
$ brew install FORMULA...
```

* 安裝相關套件

```
$ brew update
```

* 更新 Homebrew
